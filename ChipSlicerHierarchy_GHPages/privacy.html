Privacy Policy

Last updated: April 23, 2026

1. Data Collection

The ChipSlicer Hierarchy visual for Power BI does not collect, store, or transmit any personal information, user data, or business report data outside of your Power BI environment.

2. Data Processing

All data processing is performed locally within the user's Power BI infrastructure. The visual only uses data provided by the Power BI platform to render the hierarchical filter elements (chips/pills) across up to three levels (Category → SubCategory → Product).

3. Third-Party Services

This visual does not make calls to external servers and does not use any third-party services. It strictly complies with Microsoft's security policies for certified visuals.

4. Licensing

License management is handled exclusively through Microsoft's official Power BI licensing API (IVisualLicenseManager). No external authentication or payment processing is performed by this visual.

5. Children's Privacy

This visual is designed for business intelligence use by adults. We do not knowingly collect personal information from children under 13.

6. Contact

If you have any questions about this privacy policy, please contact us at: support@tcviz.com
